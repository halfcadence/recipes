Half Cadence

A concept for a small, walkable Seattle canelé bakery + slow pour-over bar — built on the owner's existing Risograph identity and recipe library. A side-project plan, not a live business.

A half cadence is a phrase that stops on the dominant — unresolved, hanging, pulling you toward the next. So is a daily sell-out drop.

Contents

  • plan.md — the full business + UX/design plan. Web-verified numbers. Two formats (Lean kiosk ~$150k / Full café ~$375k), an honest "profitability is a demand bet" flag, market/competition, location, unit economics, the espresso + hand pour-over + batch beverage program, brand, floor plan, packaging, and a "Gaps & Next Experiments" section.
  • farmers-market-playbook.md — the first move before any lease: a 6–8 weekend market pop-up to validate demand/price/attach/waste for ~$3,900. Opens with the WA TCS / cottage-food reality (a canelé must be baked in a commissary under a temporary permit, not a home kitchen — with the cheap lab-test path to maybe open a home channel).
  • brand/ — the visual kit: logo, the daily "today" board (the hero artifact), the circular stamp/avatar, the cup band, and the four all-star flavor glyphs. All in the two-ink riso system. See brand/README.md.
  • site/ — a small website (plain HTML, riso aesthetic): index.html landing page + today.html, the live "today's flavors" board (edit one JSON object to update; renders the dot-grid + sold-out strikes). Open either file in a browser.

The shape of the plan

  1. Validate first. Farmers-market pop-up → prove people will buy canelés at $4.25–4.50 and attach a drink, before signing anything.
  2. Open Lean. ~700 SF 2nd-gen Capitol Hill space, owner bakes, espresso + hand pour-over + batch brew, 4 all-stars + rotating drop, sell out by early afternoon.
  3. Then decide on Full. Only if Lean clears break-even with a real owner wage.

Honest caveats

  • Base-case margins are 3–5× the café benchmark — they hold only under the sell-out + premium-price thesis. The downside case goes negative once the owner takes a wage. This is a demand bet.
  • Numbers are researched + verified but flagged where estimated; confirm rents, permits, and equipment quotes locally before any money moves.
  • The TCS/cottage-food conclusion is medium-confidence (rests on the canelé's water activity > 0.85); the commissary path is the safe default regardless.
