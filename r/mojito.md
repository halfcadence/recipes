## Ingredients

| | |
|---|---|
| 3 | mint leaves |
| 1/2 oz | simple syrup |
| 2 oz | white rum |
| 3/4 oz | lime juice, freshly squeezed |
| | Club soda, to top |
| | Mint sprig, for garnish |
| | Lime wheel, for garnish |

## Steps

1. Lightly muddle mint with simple syrup in a shaker.

2. Add rum, lime juice, and ice. Give a brief shake.

3. Strain into a highball glass over fresh ice (or use pebble ice and swizzle).

4. Top with club soda.

5. Garnish with a mint sprig and lime wheel.

<div class="notes" markdown="1">
## Notes

- Slap the mint sprig firmly on the back of your hand before garnishing — this releases the aromatic oils.
- Don't over-muddle the mint leaves. You want the oils, not shredded leaves in the drink.

</div>
