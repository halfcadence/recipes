## Ingredients

| | |
|---|---|
| 60ml | umeshu (plum wine) |
| | Soda water, well chilled |
| | Ice |
| | Umeboshi or lemon twist (optional garnish) |

## Steps

1. Fill a chilled highball glass with ice.

2. Pour in umeshu.

3. Top with cold soda water. Stir gently — once — to combine.

4. Garnish with a small umeboshi or lemon twist if you like.

<div class="notes" markdown="1">
## Notes

- Ratio is roughly 1:2 or 1:3 umeshu to soda. Umeshu is sweeter than whisky so it can handle more dilution.
- Choya is the standard brand. The "extra years" aged version is richer and worth it.
- For an ume whisky highball: use 45ml Japanese whisky + 15ml umeshu + soda. Best of both worlds — the whisky gives backbone, the umeshu adds sweet-tart plum flavor.
- Everything cold. Glass, umeshu, soda.
- A single umeboshi dropped in the glass looks cute and adds a salty-sour hit as you drink.
- Non-alcoholic version: use ume syrup (梅シロップ) instead of umeshu. Same ratio.
</div>
