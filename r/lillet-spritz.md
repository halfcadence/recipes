## Ingredients

### Classic

| | |
|---|---|
| 2 1/2 oz | Lillet Blanc |
| 3 1/2 oz | club soda or sparkling water |
| 1/2 | orange slice |
| | Ice |

### Sparkling (variation)

| | |
|---|---|
| 2 1/2 oz | Lillet Blanc |
| 2 oz | sparkling wine (prosecco, cava, or champagne) |
| 3 1/2 oz | club soda or sparkling water |
| 1/2 | orange slice |
| | Ice |

## Steps

1. Place ice in a wine glass.

2. Add Lillet (and sparkling wine, if making the sparkling version).

3. Add orange slice — twist it first for more orange oil.

4. Top with club soda.

## Analysis

<div class="analysis" markdown="1">

Total drink: **~200g** (classic: 2½ oz Lillet at ~17%, 3½ oz soda, over ice)

| Component | Amount | % of drink |
|---|---|---|
| Alcohol (ethanol) | ~10g | 5.0% |
| Sugar | ~4g | 2.0% |
| Water + dilution | ~186g | 93.0% |

**A low-ABV aperitif — ~5% finished, barely sweet.** Lillet is a fortified aromatised wine, not a spirit, so even a generous pour lands light; the soda and ice make up over 90% of the glass. The sparkling variation swaps in prosecco for more lift and a touch more alcohol, but the character stays session-light.

</div>

<div class="notes" markdown="1">
## Notes

- The classic version is just Lillet + soda + orange. Light and low-alcohol.
- The sparkling version adds prosecco/cava for more effervescence and body.

</div>
