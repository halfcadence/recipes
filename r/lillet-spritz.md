## Ingredients

### Classic

| | |
|---|---|
| 2 1/2 oz | Lillet Blanc |
| 3 1/2 oz | club soda or sparkling water |
| 1/2 | orange slice |
| | Ice |

### Sparkling (variation)

| | |
|---|---|
| 2 1/2 oz | Lillet Blanc |
| 2 oz | sparkling wine (prosecco, cava, or champagne) |
| 3 1/2 oz | club soda or sparkling water |
| 1/2 | orange slice |
| | Ice |

## Steps

1. Place ice in a wine glass.

2. Add Lillet (and sparkling wine, if making the sparkling version).

3. Add orange slice — twist it first for more orange oil.

4. Top with club soda.

<div class="notes" markdown="1">
## Notes

- The classic version is just Lillet + soda + orange. Light and low-alcohol.
- The sparkling version adds prosecco/cava for more effervescence and body.
- Twist the orange slice before dropping it in to release the oils.
